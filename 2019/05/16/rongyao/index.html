使用hexo+github pages搭建博客
安装要求
确保自己电脑以安装以下工具：

git
nodejs
安装步骤
官网地址： https://hexo.io/zh-cn/

hexo模板主题：https://hexo.io/themes/

1、全局安装hexo-cli

npm install hexo-cli -g
2、初始化项目

hexo init
3、hexo常用的几个命令：

hexo n ‘文章的名称’ #创建文章
hexo g #生成静态页面
hexo s #启动本地服务，进行预览
hexo s 后直接打开浏览器输入地址进行预览即可。

​

创建github pages仓库
1、要求

创建的仓库名格式必须为： 用户名.github.io

如github用户名为ww1234, 仓库名为：ww1234.github.io

2、修改 _config.yml 配置git仓库：

修改配置文件 _config.yml ，指定仓库地址
deploy:
type: git
repo: git@github.com:qingtian-boy/qingtian-boy.github.io.git
branch: master
需配置好ssh协议的私钥和公钥

安装一个工具，用于部署到仓库中
npm install hexo-deployer-git –save
网站部署到github page仓库中

hexo d
换hexo网站主题
参考地址：https://hexo.io/themes/

在创建一个仓库，用于备份博客源码
